# 🔬 Deep Research Agent

An **Agentic AI system with Multi-Agent Orchestration** that autonomously performs end-to-end research by refining user intent, planning searches, retrieving information and generating structured reports.

---

## 🚀 Live Demo

👉 https://huggingface.co/spaces/deejay14/Deep-Research-Agent

---

## ✨ Key Highlights

* 🧠 **Agentic AI Workflow** — transforms vague queries into structured insights
* 🤖 **Multi-Agent Orchestration** — specialized agents coordinated by a manager agent
* ❓ **Clarification Engine** — refines user intent through guided questions
* 🌐 **Automated Web Research** — dynamic search and information retrieval
* 📝 **Report Generation** — produces detailed, structured markdown reports
* ⚡ **End-to-End Automation** — from query → insights with minimal human input

---

## 🧩 Architecture

```
User Query
    ↓
Clarifier Agent (refines intent)
    ↓
Planner Agent (generates search strategy)
    ↓
Search Agent (retrieves information)
    ↓
Writer Agent (synthesizes report)
    ↓
Final Research Output
```

---

## ⚙️ Tech Stack

* Python
* OpenAI Agents SDK
* Gradio
* Pydantic

---

## 🧠 How It Works

* The system first **asks clarifying questions** to better understand user intent
* A planner agent generates **targeted search queries**
* A search agent performs **web-based information retrieval**
* A writer agent compiles findings into a **comprehensive report**
* A manager agent **orchestrates the entire workflow**
